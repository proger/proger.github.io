We have two datasets of disjoint discrete integer sequences \(L\) and \(R\). Alternatively, these are sentences in different languages with no common words.

We do not know correct possible translations from one sequence to another, however we would like to let the user of a system propose their rules of translation. We would like to perform joint language modeling of both languages to learn their individual regularities and learn an algorithm to rewrite sequences in one language into sequences in another from examples.

N, D, L, R = 32, 32, 100, 100
soh, stx, pad = L+R, L+R+1, L+R+2

Both is a zero shot translation example. How do I make one sequence of 5-shot examples where the last translation follows from the previous four?

Let’s design a latent vocabulary of subsequences of the dataset on the left to the subsequences of the dataset on the right. We assume left vocabulary is speech like so let’s add some speech-like properties: multiple ways to same the say thing (many to one mapping),

def make_sequence():
    # define vocabulary of source words of length 1,2,3,4 tokens
    vocabL1 = torch.randint(0, L, (1, 1))
    vocabL2 = torch.randint(0, L, (1, 2))
    vocabL3 = torch.randint(0, L, (4, 3))
    vocabL4 = torch.randint(0, L, (4, 4))

    # define parallel of target words of length 1,2 tokens
    vocabR1 = torch.randint(L, L+R, (5, 1))
    vocabR2 = torch.randint(L, L+R, (5, 2))
    
    vocabR = vocabR1.tolist() + vocabR2.tolist()
    vocabL = vocabL1.tolist() + vocabL2.tolist() + vocabL3.tolist() + vocabL4.tolist()

    # assemble source sentences
    sources = torch.randint(0, len(vocabL), (5, 3))
    source_sequences = [
        [token for word in source_sequence for token in vocabL[word]] 
        for source_sequence in sources
    ]

    # assemble target sentences by translating source
    target_sequences = [
        [token for word in source_sequence for token in vocabR[word]]
        for source_sequence in sources
    ]

    # format training sequences: 4 training examples, 1 test example
    training_sequence = [tok for s,t  in zip(source_sequences, target_sequences)
                         for tok in [soh] + s + [stx] + t] + [soh]
    return torch.tensor(training_sequence)

def make_batch():
    return torch.nn.utils.rnn.pad_sequence([make_sequence() for _ in range(32)], batch_first=True, padding_value=pad)

class LM(nn.Module):
    def __init__(self):
        super().__init__()
        self.words = nn.Embedding(L+R+3, D, padding_idx=pad)
        self.readin = nn.Linear(D, 128)
        self.rnn = nn.LSTM(128, 1024, batch_first=True)
        self.readout = nn.Linear(1024, 256)
        self.classifier = nn.Linear(256, L+R+3)

    def forward(self, inputs):
        x = self.words(inputs)
        x = self.readin(x)
        x, _ = self.rnn(x)
        x = self.readout(x)
        x = self.classifier(x)
        return x
